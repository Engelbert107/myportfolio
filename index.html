Wine Quality Classification

In this project, we use the UCI Wine Quality dataset to classify wines as lower or higher quality based on their chemical properties. We trained six models, including K-Nearest Neighbors, Random Forest, Gradient Boosting, LightGBM, and XGBoost.

The Random Forest model achieved the best results with 81% accuracy and a ROC-AUC of 0.897. Despite minor false alarms, it delivered strong precision and recall across all classes. For real-world use, we deployed the model locally with Flask to enable effective wine quality classification.

RAG-Based Chatbot for PDF Question Answering

We built an intelligent assistant that lets users query PDF documents in natural language. Using Retrieval-Augmented Generation (RAG), it extracts, cleans, and embeds text into a ChromaDB vector store, then retrieves relevant sections to generate context-aware answers with TinyLlama (1.1B).

The conversational interface, powered by Chainlit, supports interactive streaming responses. Lightweight and CPU-friendly, the system is modular and can scale to larger LLMs or additional data sources.

Question-Answering from PDF

We developed an automated solution to extract information from PDF documents and provide answers to user queries using a combination of advanced text extraction and machine learning techniques.

We utilized tools such as pdfplumber for text extraction, PyMuPDF (fitz) for image extraction, and Tesseract OCR for optical character recognition (OCR) to handle various document formats. The extracted text was then processed using a BERT-based Question-Answering model to accurately respond to user queries.
